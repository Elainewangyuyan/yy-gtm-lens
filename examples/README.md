# Public examples

Each example contains only three public deliverables:

```text
article.md
website/
report.pdf
```

Available examples:

- [Kimi](./kimi): China AI-native model/app company sample.
- [MiniMax](./minimax): listed AI company sample with public financial disclosures.
- [Glean](./glean): B2B Work AI / enterprise AI platform sample.

Live websites:

- [Open Kimi website](https://elainewangyuyan.github.io/yy-gtm-lens/examples/kimi/website/)
- [Open MiniMax website](https://elainewangyuyan.github.io/yy-gtm-lens/examples/minimax/website/)
- [Open Glean website](https://elainewangyuyan.github.io/yy-gtm-lens/examples/glean/website/)

Private section 09 files, QA notes, and intermediate build artifacts are intentionally excluded.
